# MIMEMI 过滤器 (MIMEMI Filter)

#### 介绍

该工具主要针对 **MIMEMI** 服务提供的的订阅内容和格式进行处理，去除不希望出现的服务节点。

#### 原理

通过中间服务进行重新订阅，中间服务获取原始配置文件，根据要求对原始内容进行处理并返回新配置，以实现过滤效果。

#### 支持的格式 

  - SSR
  - SSR 995 端口
  - ClashR
  - SS
  - Clash
  - Surge 2/3
  - Surfboard

#### 暂不支持

  - SSD
